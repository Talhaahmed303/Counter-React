🚀 React Counter App

A simple yet interactive Counter Application built using React.js and Vite.
This project demonstrates state management using the useState Hook with proper boundary controls.

📌 Features

✅ Increment counter (Maximum limit: 20)
✅ Decrement counter (Minimum limit: 0)
✅ Prevents counter from going below 0
✅ Prevents counter from exceeding 20
✅ Clean and minimal UI
✅ Functional component using React Hooks

🛠️ Technologies Used
React.js
Vite
JavaScript (ES6)
CSS

🧠 Concepts Practiced
React Functional Components
useState Hook
Event Handling
Conditional Rendering
Updating State with Previous State

⚡ Installation & Setup

Clone the repository:

git clone https://github.com/your-username/react-counter-app.git

Navigate to the project directory:

cd react-counter-app

Install dependencies:

npm install

Start the development server:

npm run dev
